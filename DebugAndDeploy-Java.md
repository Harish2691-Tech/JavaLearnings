Packaging java applications: 

Package your code into a jar file. jar- Java archive 

To create a jar file : 

File -> Project structure -> new popup opened with Artifacts -> 

Artifacts - is combination of all our projects assets that we put together.

Artifacts -> + -> select the appropriate artifacts-> here JAR -> select Modules with Dependencies

Create JAR popup opened.

in that select the Main class. in a project we can have multiple classes. so we need to select the main class and click ok button.
Then in Artifacts the new jar file is added. 


To build the JAR file :

Click Build -> Build artifacts -> small popup will open with lists the jar file. 
select the jar file and click the build. (build, rebuild and clean)
...
in the project panel, we can the out folder. 
out -> artifacts -> Helloworld_jar -> HelloWorld.jar file -> right click on the file -> open in terminal.
It will opened in terminal. -> To run the file
command :
java -jar HelloWorld.jar

it will run the application in Java run time environment
